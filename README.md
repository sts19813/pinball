# Neo Pinball

Un pinball web de mesa vertical inspirado en máquinas arcade clásicas, hecho con Phaser 3 y física propia para pinball.

## Comandos

```bash
npm install
npm run dev
```

## Controles

- `A` o flecha izquierda: paleta izquierda
- `D` o flecha derecha: paleta derecha
- `Space` o `Enter`: cargar y soltar lanzador
- `R`: reiniciar partida
- En pantallas táctiles: toca el lado izquierdo o derecho para activar las paletas
